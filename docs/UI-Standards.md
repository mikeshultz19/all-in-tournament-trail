# UI Standards

- Use black and charcoal backgrounds.
- Use red headings.
- Use gold accents sparingly.
- Use white and muted gray body text.
- Use thin dividers.
- Keep card use minimal.
- Maintain clean spacing.
- Use strong typography.
- Use the shared `PageHeader` on major pages.
- Keep schedule rows simple and typography-driven.
- Mobile responsiveness is required.
- Accessibility and readable contrast are required.
